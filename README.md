# Mobile Sales Performance Dashboard (Power BI)

## 📊 Overview
This project analyzes mobile sales data to track performance, customer behavior, and sales trends using Power BI.

## 🚀 Key Features
- Multi-page dashboard (Overview, MTD Report, Same Period Last Year)  
- Time-based analysis (MTD, monthly, yearly trends)  
- City-wise sales visualization (map)  
- Payment method and brand performance breakdown  

## 🛠 Tools Used
- Power BI  
- DAX  
- Power Query  

## 📈 Insights
- Tracked sales growth over time (MTD trends)  
- Compared current vs previous year performance  
- Analyzed customer ratings and transaction patterns
